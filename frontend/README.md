# 💻 AppDeGestion - Frontend Angular (Base de travail)

Ce module est actuellement une **toile vierge** (blank canvas). L'architecture est initialisée et prête à recevoir les premiers composants UI.

---

## 🎯 Objectif de cette phase

L'objectif est d'utiliser la structure unitaire d'Angular pour construire chaque vue étape par étape :
1.  **Header & Footer** : Bases de la navigation.
2.  **Pages d'Auth** : Connexion au backend.
3.  **Dashboards** : Intégration des fonctionnalités de gestion.

---

## 🛠️ Stack Technique

*   **Framework** : [Angular 19/20](https://angular.dev/)
*   **Styles** : Tailwind CSS
*   **Statut** : Prêt pour `ng generate`

---

## 🚀 Installation & Lancement

```bash
npm install
npm start
```
Accès à la structure vide sur `http://localhost:4200`.
