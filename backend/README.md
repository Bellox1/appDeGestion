# ⚙️ AppDeGestion - Backend API

**Focus actuel : Sécurité et Authentification**

Le backend est construit sur **Laravel 12** avec une priorité mise sur un système d'authentification robuste.

---

## ✅ Ce qui est opérationnel

*   **Système d'authentification** : Inscription, connexion, et sécurisation des routes.
*   **Migration de base** : Schémas de base de données prêts pour l'évolution futur.
*   **Setup Environnement** : Prêt pour le développement de la logique métier.

---

## 🛠️ Stack Technique

*   **Framework** : [Laravel 12](https://laravel.com/)
*   **Auth** : Laravel Breeze / Fortify
*   **Base de données** : SQLite

---

## 🚀 Installation

```bash
composer install
npm install
php artisan migrate --seed
php artisan serve
```
