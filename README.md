# 🏢 Plateforme de Gestion d'Événements & Salons (Work In Progress)

**État du Projet :** Phase Initiale 🛠️
Ce projet est en cours de développement. L'objectif actuel est de construire une base solide en commençant par l'infrastructure et la sécurité.

---

## 🏗️ État de l'Architecture

1.  **Backend (Laravel 12)** : 
    *   **Statut** : Structure de base prête.
    *   **Implémenté** : Système d'authentification complet et sécurisé.
    *   **À venir** : CRUD pour les événements, stands et participants.

2.  **Frontend (Angular 20)** :
    *   **Statut** : Environnement initialisé.
    *   **Implémenté** : Architecture de base sans vues (prêt pour la conception).
    *   **À venir** : Création des composants UI et intégration de l'authentification.

---

## 🚀 Lancer le projet dans son état actuel

### Backend
```bash
cd backend
# Authentification déjà configurée
php artisan serve
```

### Frontend
```bash
cd frontend
# Projet vide, prêt pour ng generate component
npm start
```
